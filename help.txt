AI Vision - Quick Run Steps

Prerequisites
- Node.js 16+

Install dependencies
- In this folder:
  npm install

Backend (Groq)
1) Copy env template and set your key:
   copy server/.env.example server/.env
   # then edit server/.env and set:
   # GROQ_API_KEY=your_groq_api_key_here
   # PORT=8788
2) Start backend:
   npm run start:server
   # Health check: http://127.0.0.1:8788/health
   # Q&A endpoint: POST http://127.0.0.1:8788/qa

Frontend
1) Start a static server:
   npx --yes live-server --port=5173 --no-browser
2) Open in browser:
   http://127.0.0.1:5173
3) Upload an image, click Analyze, then use the bottom-right Q&A box.

Notes
- The frontend is configured to call http://127.0.0.1:8788/qa (see index.html).
- To change ports:
  - Backend: edit server/.env (PORT)
  - Frontend: update window.TEXT_MODEL_ENDPOINT in index.html

Troubleshooting
- Backend not responding: visit http://127.0.0.1:8788/health
- 401/500 from /qa: set GROQ_API_KEY in server/.env and restart backend
- CORS errors: backend has CORS enabled by default
